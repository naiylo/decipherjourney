Caesar Verschlüsselung

Die Caesar-Verschlüsselung ist eine einfache Verschlüsselungstechnik, bei der jeder Buchstabe im Klartext um eine bestimmte Anzahl verschoben wird. Zum Beispiel wird bei einer Verschiebung um 3 aus A ein D, B wird E, X wird A und Z wird C.

A → D B → E C → F

Erklärung:


Im Notizen-Bereich können Sie die Buchstaben des Caesar-Verschlüsselten Textes anpassen. Schreiben Sie einfach einen Buchstaben zwischen A-Z in das Feld neben dem Buchstaben und drücken Sie den "Try out your combination" Button. So können sie sehen wie der Text aussieht wenn bestimmte Buchstaben mit anderen ersetzt wurden und dadurch auf die Verschiebung schlussfolgern. Leere Buchstabenfelder werden durch ein _ dargestellt. Der "Clear" Button löscht alle bis jetzt aufgestellten Vermutungen von Buchstaben Paaren. Mit dem "Shift By" Button können sie direkt eine Verschiebung ausprobieren um einen festen Wert ausprobieren.

Unter verschlüsselter Text können sie den verschlüsselten Text sehen. Hier wird auch die Lösung angezeigt sollten sie sich dafür entscheiden aufzugeben und den "Reveal the original Text" Button zu klicken. Mit dem "Tryout a new cipher" Button können sie eine komplett neu verschlüsselte Nachricht erhalten.
